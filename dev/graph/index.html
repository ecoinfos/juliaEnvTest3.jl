Adding plots in md file

  • By using @example name as sampleGraph we can combine codes. So with one declaration of using statement, we use it repeatedly for all subsequent plots.
  • Without save or savefig, we can still add plots but to be more explicit, we can define save the fig file and position it by ![](filename).
  • ans variable is defined by Documenter for last command. So, without repeating plot command, we can use it to save file.
