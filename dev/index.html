juliaEnvTest3.sampleMod.addone — Function
addone(x, y=1)

Compute plus x and y.

If y is unspecified, compute add by 1 using default y value as 1.

Examples

julia> addone(1)
2

Arguments

  • x::Integer: first number to add.
  • y::Integer=1: second number to add.
